✦ Enable Short Links via Supabase
  1. Create a free account at supabase.com (no credit card)
  2. New project → SQL Editor → run:
    create table flows (id uuid primary key default gen_random_uuid(), data jsonb not null, created_at timestamptz default now());
  3. Project Settings → API → copy Project URL and anon public key
  4. Paste both into the SUPABASE_URL and SUPABASE_ANON_KEY variables at the top of the HTML file
  5. In Supabase → Authentication → Policies, enable public insert and select on the flows table
